📖 문법 가이드

단어를 외울 때 어미(뒷부분)만 갈아끼우면 시제·문체가 바뀝니다. 아래 규칙을 알면 활용이 훨씬 쉬워져요.

1 · 활용의 기본 규칙
🏃 동사 — 가장 규칙적

공손체는 ます만 변신, 보통체 부정은 ない 계열.

🎨 い형용사 — い를 갈아끼운다

い를 떼고 → 부정은 く, 과거는 かった. 공손체 = 보통체 + です.

🎯 핵심 3가지
  • 부정의 시작으로 품사를 가른다 — 동사·い형용사는 나/쿠, 명사·な형용사는 쟈
  • 과거는 "타(た)" 소리로 끝남 — 마시타 / 데시타 / 캇타
  • 과거부정 = 부정형 + 한 번 더 과거로 (나이→나캇타)
2 · 동사 て형 (연결형)

て형 하나면 "~하고 있다 / ~해 주세요 / ~해도 된다"가 다 됩니다. ~하고, ~해서라는 뜻.

3 · 쉽게 문장 만들기

일본어는 "누가 + 무엇을 + 어떻게" 순서. 동사·형용사가 맨 뒤에 옵니다.

🧱 기본 문장 골격
패턴예
A는 B입니다
A は B です		私は学生です
와타시와 가쿠세이데스 · 저는 학생입니다
~을 ~합니다
~を ~ます		ご飯を食べます
고한오 타베마스 · 밥을 먹습니다
~에 갑니다
~に 行きます		学校に行きます
갓코니 이키마스 · 학교에 갑니다
~에서 ~합니다
~で ~ます		家で休みます
이에데 야스미마스 · 집에서 쉽니다
🔗 핵심 조사 (단어를 잇는 접착제)
조사역할예
は (와)~은/는 (주제)私は · 저는
が (가)~이/가 (주어)雨が · 비가
を (오)~을/를 (목적)水を · 물을
に (니)~에 (방향·시간)駅に · 역에
で (데)~에서 (장소)店で · 가게에서
と (토)~와/과友達と · 친구와

💡 は는 조사로 쓰일 때 "하"가 아니라 "와"로 읽어요. を도 "오".

✏️ 문장 확장 연습
  • 食べます (먹습니다) → ご飯を食べます (밥을 먹습니다)
  • → 家でご飯を食べます (집에서 밥을 먹습니다)
  • → 友達と家でご飯を食べます (친구와 집에서 밥을 먹습니다)

조사로 단어를 하나씩 이으면 문장이 자연스럽게 길어집니다.
